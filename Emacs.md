## Spostare il point
### Spostare il point di un carattere

```
C-f 
C-b
C-n
C-p
```

### Inizio di una linea

```
C-a
```

### Fine di una riga

```
C-e
```
### Sposta di una parola

```
M-f
M-b
```

### Sposta il point al centro del testo poi all'inizio e poi alla fine ciclicamente

```
M-r
```
